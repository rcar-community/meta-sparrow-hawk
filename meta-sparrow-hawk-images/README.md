# meta-sparrow-hawk-images

Yocto layer providing image recipes and configurations for the Sparrow Hawk board.

## Dependencies

- meta-sparrow-hawk-bsp
- openembedded-layer (meta-oe)
- meta-python

## How to run yocto-check-layer

```bash
bash scripts/run-yocto-layer-check-images.sh
```
